# Installation ( Termux )

1. pkg update && pkg upgrade -y
2. pkg install nodejs -y
3. pkg install git -y
4. pkg install termux-api -y
5. git clone https://github.com/Boiescylosh/Wa-Clone
6. cd Wa-Clone
7. npm install
8. node setup.js
9. node Wa-Clone.js
